    • Javascript Nedir ?

    JavaScript, yaygın olarak web tarayıcılarında kullanılmakta olan bir betik dilidir. JavaScript ile yazılan istemci tarafı betikler sayesinde tarayıcının kullanıcıyla etkileşimde bulunması, tarayıcının kontrol edilmesi, asenkron bir şekilde sunucu ile iletişime geçilmesi ve web sayfası içeriğinin değiştirilmesi gibi işlevler sağlanır. JavaScript, Node.js gibi platformlar sayesinde sunucu tarafında da yaygın olarak kullanılmaktadır.

    lk olarak bir Netscape çalışanı olan Brendan Eich tarafından geliştirilen ve 1997 yılından beri bir ECMA standardı olan JavaScript, günümüzde Mozilla Vakfı öncülüğünde özgür yazılım topluluğu tarafından geliştirilmekte ve bakımı yapılmaktadır.

    Web sayfalarının görünümünün mükemmelleştirilmesi, işlevselliğinin artırılması ve dinamik web sayfaları tasarlanması amacıyla JavaScript, HTML kodu içinde gömülü olarak (embedded) kullanılabilir. JavaScript, bir programlama dili disiplini ve özelliklerine sahiptir.

    Javascript ile Java Arasındaki farklar

    Javascript

    İstemci (client) tarafından (örn. web tarayıcı) yorumlanır (derlenmez).

    Nesneye dayalı bir dildir. Nesne tipleri arasında farklılık yoktur. Herhangi bir nesneye yeni özellik ve metotlar dinamik olarak eklenir. Nesneye yönelik teorinin miras alma (inheritance) özelliği prototip mekanizmasıyla gerçekleştirilir.

    Değişken veri tip bildirimi mekanizması yoktur.

    Dinamik bağlama mekanizması kullanılır yani nesneye verilen referanslar çalışma zamanında (run time) kontrol edilir.

    Java

    Derlenmiş bytecode denilen program istemcide çalıştırılır.

    Nesneye yönelik bir dildir. Nesneler çeşitli sınıflara ayrılmıştır ve miras alma mekanizması, sınıf hiyerarşisi sayesinde gerçekleştirilir. Sınıf ve sınıf örneklerine özellik ve metotlar dinamik olarak eklenemez.

    Değişken tip bildirimi zorunludur.

    Statik bağlama mekanizması kullanılır yani nesne referansları derleme zamanında (compile time) kontrol edilir.
